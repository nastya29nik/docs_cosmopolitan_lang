# Синтаксис языка Cosmopolitan

Язык использует простой LISP-подобный синтаксис с квадратными скобками `[...]`. Первый элемент в скобках обычно является функцией или ключевым словом, а остальные — её аргументами.

---

### Переменные (`let`)

Связывает имя с результатом вычисления выражения. Область видимости — выражение, следующее за `let`.

```scheme
[let x 42
    [print x]] ; Выведет: 42
```

### Рекурсия (`letrec`)

`letrec` (let recursive) позволяет функции ссылаться на саму себя в своем теле. Используется для определения рекурсивных функций.

```scheme
[letrec fact 
    [lambda n 
        [if [== n 0] 
            1 
            [* n [fact [- n 1]]]]]
    [fact 5]] ; Результат: 120
```

### Функции (`lambda`) и замыкания

Анонимные функции создаются с помощью `lambda`. Они захватывают переменные из окружения, в котором были созданы (замыкания).

```scheme
[let addN 
    [lambda n          ; Эта функция возвращает другую функцию
        [lambda x 
            [+ x n]]]   ; 'n' захватывается из внешнего окружения
    [let add5 [addN 5] ; Создаем функцию, которая прибавляет 5
        [add5 10]]]    ; Результат: 15
```

### Условные выражения (`if`)

`if` принимает три аргумента: условие, выражение для случая `true` и выражение для случая `false`.

```scheme
[if [< 3 5]
    "three is less"
    "three is not less"] ; Результат: "three is less"
```

### Списки (`list`)

Списки — это упорядоченные коллекции элементов. Создаются с помощью ключевого слова `list`.

```scheme
[let numbers [list 1 2 3 4 5]
    [let doubled [map [lambda x [* x 2]] numbers]
        doubled]] ; Результат: [2, 4, 6, 8, 10]
```

### Конвейер (`pipe`)

Оператор `pipe` передает результат левого выражения в качестве аргумента правому (которое должно быть функцией). Это аналог оператора `|>` в F#.

```scheme
[pipe [list 1 2 3]
    [lambda lst [map [lambda x [* x 2]] lst]]] ; Результат: [2, 4, 6]
```
Это эквивалентно: `[[lambda lst [map ...]] [list 1 2 3]]`.

### Ленивые вычисления (`delay` и `force`)

`delay` "заворачивает" выражение, откладывая его вычисление. `force` заставляет отложенное выражение вычислиться.

```scheme
[let expensive [delay [+ 100 200]]
    [block
        [print "Expensive not yet computed"]
        [force expensive]]] ; Вычисление `+ 100 200` произойдет только здесь
```

### Блоки кода (`block`)

`block` выполняет последовательность выражений и возвращает результат последнего.

```scheme
[block
    [print "Calculating..."]
    [let result [* 5 5]
        [print result]]
    "Done"] ; Результат всего блока: "Done"
```
