# Добро пожаловать в Cosmopolitan Lang!

**Cosmopolitan** — это функциональный язык программирования, написанный на F#. Он основан на лямбда-исчислении и поддерживает современные возможности функционального программирования: замыкания, ленивые вычисления, систему списков и монадические операции.

Сайт с документацией: [Синтаксис языка](./syntax.md) | [Стандартная библиотека](./stdlib.md) | [Инструменты](./ecosystem.md)

---

## Ключевые возможности

*   **Именованные переменные (`let`)**: Простое и понятное связывание имен со значениями.
*   **Рекурсия (`letrec`)**: Полноценная рекурсия для реализации сложных алгоритмов.
*   **Функции (`lambda`) и замыкания**: Функции как объекты первого класса с захватом лексического окружения.
*   **Ленивые вычисления (`delay`, `force`)**: Отложенное вычисление ресурсоемких операций.
*   **Списки и функции высшего порядка**: Встроенная поддержка списков с функциями `map` и `flatMap`.
*   **Конвейерная обработка (`pipe`)**: Элегантный синтаксис для цепочек вызовов функций.
*   **Ввод-вывод**: Встроенные функции для работы с консолью и файловой системой.
*   **Интерактивная Web IDE**: Пишите и запускайте код прямо в браузере.
*   **Поддержка VS Code**: Расширение для подсветки синтаксиса.

## Быстрый старт

### Требования

- .NET 10.0 SDK или выше
- F# 10.0 или выше

### Сборка и запуск

1.  **Склонируйте репозиторий и соберите проект:**
    ```bash
    git clone https://github.com/MAILabs-Edu-2026/funcpro-coursework-cosmopolitan.git
    cd funcpro-coursework-cosmopolitan
    dotnet build
    ```

2.  **Запустите Web IDE (режим по умолчанию):**
    ```bash
    dotnet run
    ```
    После этого откройте в браузере `http://localhost:5000`.

3.  **Запустите программу из файла:**
    ```bash
    dotnet run -- examples/recursion.cmpn
    ```

4.  **Запустите все встроенные тесты:**
    ```bash
    dotnet run -- test
    ```

## Использование генеративного ИИ

В процессе разработки проекта команда использовала генеративный ИИ (Qwen, Gemini) для решения следующих задач: парсинг рекурсивных выражений, реализация замыканий и рекурсии в интерпретаторе, сравнение сложных типов для тестов, создание шаблона модульного тестирования и реализация self-hosted HTTP-сервера. Промпты и результаты их внедрения описаны в основном файле `README.md` репозитория.
