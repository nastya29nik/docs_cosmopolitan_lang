# Стандартная библиотека Cosmopolitan

Язык поставляется с набором встроенных функций для выполнения базовых операций.

---

### Арифметические операторы

Все операторы каррированы: их можно применять с одним аргументом, чтобы получить новую функцию.

- `[+ a b]` — Сложение
- `[- a b]` — Вычитание
- `[* a b]` — Умножение
- `[/ a b]` — Целочисленное деление. Вызывает ошибку при делении на ноль.

*Пример:*
```scheme
[[+ 10] 5] ; Результат: 15
```

---

### Операторы сравнения

- `[== a b]` — Равно
- `[<> a b]` — Не равно
- `[< a b]` — Меньше
- `[> a b]` — Больше
- `[<= a b]` — Меньше или равно
- `[>= a b]` — Больше или равно

---

### Логические операторы

- `[&& a b]` — Логическое И
- `[|| a b]` — Логическое ИЛИ

---

### Работа со строками

- `[++ s1 s2]` — Конкатенация строк.

*Пример:*
```scheme
[++ "Hello, " "World!"] ; Результат: "Hello, World!"
```

---

### Ввод-вывод

- `[print value]` — Печатает значение в консоль и возвращает его же.
- `[readFile path]` — Читает содержимое файла по указанному пути и возвращает его как строку.
- `[writeFile path content]` — Записывает строку `content` в файл по пути `path`.

---

### Работа со списками

- `[map func list]` — Применяет `func` к каждому элементу `list` и возвращает новый список с результатами.
- `[flatMap func list]` — Аналогично `map`, но `func` должна возвращать список. Результаты всех списков объединяются в один плоский список (монадический `bind`).
- `[head list]` — Возвращает первый элемент списка. Ошибка на пустом списке.
- `[tail list]` — Возвращает список без первого элемента. Ошибка на пустом списке.
- `[isEmpty list]` — Возвращает `true`, если список пуст, иначе `false`.
- `[cons head tail]` — Добавляет элемент `head` в начало списка `tail`.

*Пример flatMap:*
```scheme
[flatMap [lambda x [list x x]] [list 1 2]] ; Результат: [1, 1, 2, 2]
```
