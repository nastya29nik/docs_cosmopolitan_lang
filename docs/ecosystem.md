# Экосистема и инструменты

Cosmopolitan — это не только язык, но и удобные инструменты для разработки.

---

## Web IDE

В проект встроена простая среда разработки, работающая прямо в браузере. Она идеально подходит для быстрого прототипирования и изучения языка.

### Как запустить?

Выполните в терминале команду:
```bash
dotnet run
```
Затем откройте в браузере адрес `http://localhost:5000`.

### Возможности

- **Редактор кода**: Пишите код на Cosmopolitan в левой панели.
- **Интерактивная консоль**: В правой панели отображаются результаты выполнения и вывод функции `print`.
- **Мгновенное выполнение**: Нажмите кнопку "▶ Run Code", и ваш код будет выполнен на сервере, а результат сразу же отобразится в консоли.

## Расширение для VS Code

Для более комфортной разработки в файлах (`.cmpn`) мы создали простое расширение для Visual Studio Code.

### Возможности

- **Подсветка синтаксиса**: Ключевые слова (`let`, `lambda`), функции (`print`, `map`), числа, строки и комментарии подсвечиваются разными цветами.
- **Автоматическое закрытие скобок**: `[` автоматически закрывается `]`.

### Как установить?

1. Скопируйте папку `vscode-cosmo` из корня проекта.
2. Вставьте её в директорию расширений VS Code. Пути для разных ОС:
   - **Windows**: `%USERPROFILE%\.vscode\extensions`
   - **macOS/Linux**: `~/.vscode/extensions`
3. Перезапустите VS Code.

Теперь все файлы с расширением `.cmpn` будут автоматически подсвечиваться.
